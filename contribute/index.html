Contribute

This wiki is built with Hugo and the source code is hosted at github.com/SynthDIY/wiki. The site is hosted on github pages and gets automatically updated when a commit is pushed to the main branch.

To contribute you can either send a pull request or contact us on Github or Discord and ask to be invited to the Github organization.

Repository structure and article formatting

All the content in the wiki lives in the /content folder of the repository and consists of pages written in markdown with a little bit of extra metadata.
